# 📊 Topological Index Calculator

Bu uygulama, grafların topolojik indekslerini (özellikle **Wiener İndeksi**) hesaplamak ve görselleştirmek için geliştirilmiştir.

## 🚀 Canlı Siteye Git
Uygulamayı tarayıcınızda denemek için aşağıdaki bağlantıya tıklayın:
👉 **[Topological Index Calculator - Canlı Uygulama](https://topological-index-calculator-int384hivgnxjbt8uywk9f.streamlit.app/)**

## ✨ Özellikler
* **Graf Analizi:** Düğümler ve kenarlar ekleyerek graf yapısı oluşturma.
* **Matematiksel Hesaplama:** Wiener İndeksi ve Fibonacci/Lucas küpü analizleri.
* **Görselleştirme:** Matplotlib ile interaktif grafik çizimi.

## 🛠️ Nasıl Çalışır?
1. Sol menüden graf tipini seçin.
2. Parametreleri girin.
3. Hesapla butonuna basarak topolojik değerleri görün.
